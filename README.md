# Сайт "Лучшие Хлебобулочные Изделия"

Адаптивный веб-сайт о хлебобулочных изделиях, разработанный как учебный проект.

## Структура проекта

```
bakery-website/
│
├── index.html          # Главная страница
├── about.html          # Страница "О нас"
├── products.html       # Страница с продукцией
├── tasks.html          # Страница для заданий
├── css/
│   └── style.css       # Основные стили
├── images/             # Папка для изображений (опционально)
└── README.md           # Этот файл
```

## Особенности

✅ Адаптивный дизайн (работает на мобильных, планшетах и десктопах)
✅ Семантическая разметка HTML5
✅ Логотип создан только средствами CSS
✅ Flexbox и CSS Grid для макетирования
✅ Более 300 слов текста и 5+ изображений на главной странице
✅ Единый стиль на всех страницах
✅ Header, Footer, Навигация на всех страницах

## Как открыть сайт локально

1. Скачайте все файлы проекта
2. Откройте файл `index.html` в браузере
3. Или используйте локальный сервер:
   - Установите расширение "Live Server" в VS Code
   - Кликните правой кнопкой на index.html → "Open with Live Server"

## Как разместить на GitHub Pages

### Шаг 1: Создайте репозиторий на GitHub

1. Зайдите на [GitHub](https://github.com)
2. Нажмите "New repository"
3. Введите название, например: `bakery-website`
4. Выберите "Public"
5. Нажмите "Create repository"

### Шаг 2: Загрузите файлы

**Способ 1: Через веб-интерфейс GitHub**
1. На странице репозитория нажмите "Add file" → "Upload files"
2. Перетащите все файлы проекта
3. Нажмите "Commit changes"

**Способ 2: Через Git (если установлен)**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ваш-username/bakery-website.git
git push -u origin main
```

### Шаг 3: Включите GitHub Pages

1. В репозитории перейдите в "Settings"
2. В левом меню выберите "Pages"
3. В разделе "Source" выберите ветку "main"
4. Выберите папку "/ (root)"
5. Нажмите "Save"
6. Подождите 1-2 минуты

Ваш сайт будет доступен по адресу:
`https://ваш-username.github.io/bakery-website/`
