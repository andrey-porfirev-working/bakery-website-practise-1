Раздел для заданий

Эта страница предназначена для размещения учебных заданий по курсу веб-разработки. Здесь будут размещены все практические задания и курсовая работа.

  • Задание 1: Создание адаптивного макета сайта

    Разработать базовую структуру сайта с использованием HTML5 семантических тегов.

  • Задание 2: Стилизация с помощью CSS

    Применить стили к элементам сайта, создать адаптивный дизайн с использованием CSS Grid и Flexbox.

  • Задание 3: Создание логотипа средствами CSS

    Разработать уникальный логотип используя только CSS без использования изображений.

  • Задание 4: Адаптивная навигация

    Создать адаптивное меню навигации, которое корректно отображается на различных устройствах.

  • Курсовая работа: Полноценный веб-сайт

    Разработка многостраничного адаптивного сайта с логической и физической структурой, наполненного контентом и готового к размещению на хостинге.
